# Mi Media Naranja

Página web romántica creada para recordar momentos especiales en pareja.

## Archivos incluidos

- `index.html`: página principal.
- `assets/`: carpeta con las imágenes del sitio.

## Cómo publicarla en GitHub Pages

1. Entra a tu repositorio `MiMediaNaranja`.
2. Sube `index.html` y la carpeta `assets`.
3. Ve a **Settings → Pages**.
4. En **Source**, selecciona **Deploy from a branch**.
5. En **Branch**, selecciona `main` y `/root`.
6. Guarda con **Save**.
7. Espera unos minutos.

Tu sitio debería quedar en:

```txt
https://kalef777.github.io/MiMediaNaranja/
```

## Nota

Si cambias el nombre del repositorio o el usuario de GitHub, la URL también cambia.
